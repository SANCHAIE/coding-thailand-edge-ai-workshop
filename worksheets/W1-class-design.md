## 1. โจทย์ของทีม

**Problem statement** (1-2 ประโยค บอกว่าจะแก้ปัญหาอะไรในชีวิตจริง):

```
ทีมของเราจะสร้างระบบคัดแยกขยะอัตโนมัติด้วยปัญญาประดิษฐ์ (AI)
โดยใช้การตรวจจับภาพ (Vision AI) เพื่อจำแนกประเภทขยะออกเป็น 4 ประเภท
ได้แก่ ขยะทั่วไป ขยะรีไซเคิล ขยะเปียก และขยะอันตราย
เพื่อช่วยลดการคัดแยกผิดประเภทและส่งเสริมการจัดการขยะอย่างถูกต้อง
```

## 2. Class Definition

ระบุ class ที่จะใช้ + เกณฑ์ที่ทำให้แต่ละ class แตกต่าง:

| # | ชื่อ Class | คำอธิบาย (3-4 ประโยค) | สิ่งที่ทำให้ต่างจาก class อื่น |
 1 | ขยะทั่วไป | เป็นขยะที่ไม่สามารถนำกลับมาใช้ใหม่ได้ง่าย เช่น ซองขนม โฟม หรือเศษวัสดุที่ปนเปื้อน พบได้บ่อยในชีวิตประจำวัน | วัสดุหลากหลาย รีไซเคิลได้ยาก |
| 2 | ขยะรีไซเคิล | เป็นขยะที่สามารถนำกลับมาแปรรูปใช้ใหม่ได้ เช่น ขวดพลาสติก กระดาษ และกระป๋องอลูมิเนียม | มีวัสดุที่นำกลับมาใช้ใหม่ได้ |
| 3 | ขยะเปียก | เป็นขยะอินทรีย์ที่ย่อยสลายได้ เช่น เศษอาหาร เปลือกผลไม้ และเศษผัก | มีความชื้นสูง และย่อยสลายได้ |
| 4 | ขยะอันตราย | เป็นขยะที่มีสารเคมีหรือวัตถุอันตราย เช่น ถ่านไฟฉาย หลอดไฟ และกระป๋องสเปรย์ | อาจเป็นอันตรายต่อสุขภาพและสิ่งแวดล้อม |

**ตัวอย่าง (Track A, Fall Detector):**

| # | ชื่อ Class | คำอธิบาย | สิ่งที่ทำให้ต่าง |
|---|---|---|---|
| 1 | ยืน | ผู้ใช้ยืนนิ่ง หรือยืนเฉยๆ ใส่ที่เอว | accel แกน Z สูง, gyro ต่ำ |
| 2 | เดิน | เดินปกติ ก้าวเท้าสลับ | accel oscillation, periodic |
| 3 | ล้ม | ล้มลงพื้นกะทันหัน | high accel spike + horizontal Z |

## 3. Edge Cases — สิ่งที่อาจสับสน

ระบุสถานการณ์ที่อาจทำให้โมเดลสับสน:

```
1. ขวดน้ำที่ยังมีเศษอาหารอยู่อาจถูกจัดเป็น "ขยะเปียก"

2.  กล่องอาหารเปื้อนน้ำมันอาจถูกจัดเป็น "ขยะทั่วไป" แทน "ขยะรีไซเคิล"

3.กระป๋องสเปรย์ที่มีลักษณะคล้ายกระป๋องน้ำอัดลมอาจถูกจัดเป็น "ขยะรีไซเคิล"
```

**วางแผนแก้:**
```
เพิ่มรูปภาพใน dataset ให้หลากหลายมากขึ้น
เก็บภาพจากหลายมุม หลายสภาพแสง และหลายลักษณะของขยะ
เพื่อช่วยให้โมเดลเรียนรู้ความแตกต่างของแต่ละประเภทได้แม่นยำขึ้น
```

**คาดว่าใช้เวลาเก็บ:** 40-60 นาที

⚠️ **เป้าหมายขั้นต่ำ:**
- Motion/Audio: 50 samples × class
- Vision: 100 images × class
- Environment: 200 samples × class

## 5. Diversity Plan — กันอคติ (Bias)

ตอบ 3 คำถาม:

### a) มีหลาย "environment" ไหม?
```
มีการเก็บข้อมูลจากหลายสถานที่ เช่น ห้องเรียน โรงอาหาร และจุดคัดแยกขยะ
รวมถึงเก็บในหลายสภาพแสง
```

### b) มีหลาย "subject" ไหม? (กรณี Vision/Audio/Motion)
```
ใช้ขยะจากหลายแหล่งและหลายรูปแบบ
รวมถึงใช้พื้นหลังที่แตกต่างกันในการถ่ายภาพ

```

### c) มีหลาย "variation" ไหม?
```
มีการถ่ายภาพหลายมุม หลายระยะ และหลายลักษณะการวางขยะ
เช่น วางตั้ง วางนอน หรือมีรอยเปื้อนแตกต่างกัน
```
